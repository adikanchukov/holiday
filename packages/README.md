#Package format

##General
Text file with the same structure as below:

*\<name\>;\<year\>*

*\<description\>*

*\<date\>;\<name\>*

*\<date\>;\<name\>*

...

##Field formats
**name**: *text*

**year**: *YYYY*

**description**: *text*

**date**: *ddMM*

**name**: *text*

##Example

GER;2016

German holiday package

0101;New Year's Day
